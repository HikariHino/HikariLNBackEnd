# Simple HTML Site

This project is a simple HTML website that demonstrates the basic structure of a web application. It includes HTML, CSS, and JavaScript files, along with directories for assets such as fonts and data.

## Project Structure

```
simple-html-site
├── src
│   ├── index.html        # Main HTML document
│   ├── css
│   │   └── styles.css    # Styles for the website
│   ├── js
│   │   └── main.js       # JavaScript for interactivity
│   └── assets
│       ├── fonts         # Directory for font files
│       └── data          # Directory for data files
├── .gitignore            # Git ignore file
└── README.md             # Project documentation
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `src/index.html` file in a web browser to view the website.
3. Modify the `src/css/styles.css` file to change the styles as needed.
4. Update the `src/js/main.js` file to add interactivity.

## Usage

This project can be used as a starting point for building more complex web applications. You can expand upon the existing structure by adding more HTML pages, styles, and JavaScript functionality as required.
